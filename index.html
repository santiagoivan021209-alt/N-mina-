Conexión a Google Sheets

Todos los datos se guardan en Google Sheets a través de un Apps Script. Si querés conectar una planilla nueva (o cambiar la actual), pegá la URL del script acá abajo.
⚠ Al cambiar la URL los datos se cargarán desde la planilla nueva.

Planilla actualmente conectada
Conectar nueva planilla
Cómo obtener la URL de una planilla nueva
1. Abrí la planilla en Google Sheets
2. Menú Extensiones → Apps Script
3. Pegá el mismo código que usaste la primera vez y guardá
4. Clic en Implementar → Nueva implementación → Aplicación web
5. En "Quién tiene acceso" elegí Cualquier usuario, luego implementá
6. Copiá la URL que termina en /exec y pegala arriba
